Módulos del Sistema
A
📲
Capa de Entrada
Canal de comunicación con el usuario
▾
Canal
WhatsApp Business API via Twilio. Webhook que recibe mensajes entrantes.
Twiliowebhook
Autenticación
Número de teléfono registrado del docente. Lista blanca en base de datos.
authseguridad
Formato de entrada
Texto libre. Sin formato obligatorio. Ej: "Hoy todos presentes excepto María".
NL input
B
🧠
Capa de IA / Procesamiento
Interpretación y estructuración del lenguaje natural
▾
Motor NLP
Python + spaCy para NER (nombres). LLM (GPT-3.5 / local) para comprensión semántica compleja.
spaCyLLMNER
Lógica de parsing
Extrae: alumno, estado (presente/ausente), fecha, observación. Confidence score para validación.
parsingconfidence
Fallback
Si confidence <80%: solicita confirmación al docente. Respuesta: "¿Confirmas: Pedro ausente, resto presentes?"
UXvalidación
C
🗄️
Capa de Integración CRM
Escritura y lectura de datos en Salesforce
▾
API Salesforce
REST API con OAuth 2.0. Crea/actualiza objetos Custom: Asistencia__c, Alumno__c.
RESTOAuth
Salesforce Flows
Flow automático al recibir registro: notifica coordinador si alumno acumula 3 ausencias.
Flowalerta

📈

Power BIKPIs

ROIKPI

Docentes con solo celular. Conectividad limitada. Resistencia a apps nuevas. Lenguaje informal.

stackbajo costo

🔨
